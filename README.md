# 개인정보처리방침 에디터

개인정보처리방침 문서를 고치기 위한 전용 웹 에디터. 기존 방침 원본 Word(`.docx`) 또는 이전에 저장해둔 구조화 데이터(`.json`)를 불러와 화면 안에서 바로 수정하고, 원하는 형식으로 다시 내려받을 수 있습니다. 수정 결과가 실제 배포 화면에서 어떻게 보일지 우측 미리보기에 즉시 나타나므로 감으로 작업하지 않아도 됩니다.

모든 작업은 **브라우저 안에서만** 이뤄집니다. 파일이 서버로 업로드되거나 외부로 전송되지 않으므로 민감한 문서라도 안심하고 다룰 수 있습니다.

## 기본 작업 순서

1. 툴바 왼쪽 **파일 열기…** 를 누르거나 파일을 드래그 드롭. `.docx` 는 섹션·문단·표로 자동 분해되고, `.json` 은 이전에 저장해둔 모양 그대로 복원됩니다.
2. 좌측 편집 패널에서 수정 → 우측 미리보기 실시간 반영.
3. 툴바 오른쪽 **저장 버튼** 또는 **▾ 드롭다운** 에서 출력 형식 선택: JSON / DOCX / Markdown / 인쇄·PDF.

JSON·DOCX·Markdown 은 동일한 문서 데이터를 서로 다른 포맷으로 내보내는 것뿐이며, `.json` 과 `.docx` 는 다시 읽어 와 이어서 편집할 수 있습니다.

## 주요 기능

- 좌우 분할: 좌 편집기 / 우 미리보기 (실시간 동기화)
- 문서는 **섹션(조)** 단위로 나뉘고, 각 섹션 안에는 **라인** (문단·표 각 1개)이 순서대로 존재. 섹션·라인 모두 이동·복제·삭제 버튼 제공.
- 네 가지 라인 종류 추가 버튼:
  - **+ 텍스트** — 일반 문단. 번호(`1.` `1)` `①`)는 원본 스타일 보존을 위해 자동 통일하지 않음.
  - **+ HTML/강조** — `<b>` `<s>` 등 꾸밈 태그 포함 문단.
  - **+ 표(상단 헤더)** — 첫 줄이 열 제목, 그 아래가 데이터인 표.
  - **+ 표(좌측 라벨)** — 왼쪽 열이 라벨, 오른쪽이 값인 표.
- 셀 조작 (셀 우상단 패널):
  - **L** — 일반 셀 ↔ 라벨 셀 토글 (시각 강조 + HTML 의미론적 헤더 선언, 스크린리더 대응).
  - **r / c 숫자** — 행·열 병합. `1` 또는 `×` 로 복원. 격자 일관성 깨는 변경은 자동 차단.
- 입출력

  | 포맷 | 열기 | 저장/내보내기 |
  |------|:----:|:-------------:|
  | JSON | ✅ | ✅ |
  | DOCX | ✅ | ✅ |
  | Markdown | ❌ | ✅ |

- DOCX 업로드 시 `제N조 (제목)` 섹션 자동 인식, 표 rowspan/colspan/첫 행 thead 승격.
- 인쇄 시 편집기 숨기고 미리보기만 출력.

## 검증·경고

문서를 불러오면 미리보기 상단에 **요약 배너** 가 뜨고, 편집 중에는 400ms debounce 로 실시간 재검증됩니다. 배너 색만 봐도 상태를 알 수 있습니다.

- **파란 배너 — 자동 정리됨 (AUTO-FIX)**: 파서 또는 내보내기 시 자동 일관화. 개입 불필요.
  - 스마트따옴표 정규화, 섹션 번호 공백 정상화, 더블 스페이스 축약, 표 첫 행 thead 승격, 변경추적 `<s>` 보존, 헤딩 bold 정상화.
- **노란 배너 — 검토 필요 (REVIEW)**: 사용자 판단 필요.
  - 섹션 번호 누락/중복, 빈 섹션, 취소선 잔존, 번호 스타일·도트 문자 혼재, 스마트따옴표 잔존, 표 격자 불일치.

각 REVIEW 항목 옆의 `§3·L2` 같은 **ref 칩** 을 클릭하면 해당 위치로 자동 스크롤 + flash 강조. 문제 구간을 직접 찾아다니지 않아도 됩니다.

## 운용 팁

원본 Word 문서의 포맷이 정형화되어 있지 않아 파싱이 애매한 경우가 있습니다. 이럴 땐 한 번 에디터로 열어 JSON·DOCX 로 다시 받아둔 파일을 **"기준 파일"** 로 쓰면, 다음 개정 때 동일 이슈가 반복되지 않습니다.

## 개발

```bash
pnpm install
pnpm dev           # 개발 서버
pnpm build         # dist/ 생성
pnpm preview       # 빌드 결과 로컬 확인
```

패키지 매니저는 pnpm 10.30.3 (`package.json` `packageManager` 필드로 고정).

## 스키마

```ts
type Doc = {
  headers: string[];
  contents: { title: string; lines: string[] }[];
};
```

`line` 종류: `table` / `html` / `text` / `empty`.
